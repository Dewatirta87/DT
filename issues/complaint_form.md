## Overview
Add a complaint and suggestion submission form for Dinas Pembekalan TNI Angkatan Laut (Navy Supply Department).

## Description
This is an HTML-based form for collecting complaints, suggestions, and integrity reports. The form includes:
- Full name input
- Email (optional)
- Phone/WhatsApp number
- Type of submission (Service suggestion, Service complaint, Integrity/Corruption report)
- Subject line
- Detailed message area

## Features
- WhatsApp integration for direct contact
- Success message confirmation
- Form submission handling
- Responsive design with Navy blue color scheme
- Indonesian language interface

## Files
- HTML form with embedded CSS and JavaScript
- WhatsApp API integration (0813 4530 7924)
